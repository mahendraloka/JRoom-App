# JRoom - Aplikasi Booking Ruangan Gedung J

JRoom adalah aplikasi mobile berbasis Android yang dirancang untuk mempermudah civitas akademika dalam melakukan pemesanan (booking) ruangan di Gedung J kampus. Proyek ini dibuat sebagai tugas kuliah untuk memenuhi kebutuhan peminjaman ruangan agar lebih terstruktur dan menghindari jadwal bentrok.

*Catatan: Aplikasi ini dikembangkan menggunakan Android Studio dan murni dibuat untuk pemenuhan tugas kuliah/simulasi, sehingga beberapa fungsi backend masih menggunakan data lokal atau belum di-optimize untuk skala produksi.*

## Fitur Utama
* **Cek Ketersediaan Ruangan:** Melihat daftar ruangan di Gedung J yang kosong atau sedang digunakan.
* **Form Peminjaman:** Mengisi data peminjaman seperti nama, keperluan, tanggal, dan jam penggunaan ruangan.
* **Status Booking:** Menampilkan riwayat atau status ruangan yang berhasil di-booking oleh user.

## Tech Stack & Tools
* **IDE:** Android Studio
* **Language:** Java
* **Database/Storage:** SQLite
